# JIPS

JIPS is an implementation of MIPS in Rust. It's dumb simple.

- [ ] Tokenize language
- [ ] Parse language
- [ ] Interpreting in VM
- [ ] Interpret Byte Code
- [ ] Compile to machine
- [ ] Support profiling and observability
- [ ] GUI Environment
- [ ] Transpile to and run in [Shadow Emulator](https://github.com/Ymit24/luna)
